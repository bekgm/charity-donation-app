# Screenshots

This directory contains screenshots of the application features.

## Required Screenshots

1. **campaigns.png** - Home page showing campaign listing with filters
2. **login.png** - Login page
3. **register.png** - Registration page
4. **dashboard.png** - User dashboard with profile and donations
5. **admin.png** - Admin panel for managing campaigns
6. **create-campaign.png** - Campaign creation modal

## How to Add Screenshots

1. Run the application locally
2. Take screenshots of each page
3. Save them in this directory with the names above
4. Screenshots should be in PNG format
5. Recommended size: 1280x720 or higher
